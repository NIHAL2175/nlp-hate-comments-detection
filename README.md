## 📖 About the Project

This project uses **Natural Language Processing (NLP)** and **Machine Learning** to automatically detect **hate comments** from textual data.
It helps in identifying harmful content in online platforms.

---

## ✨ Features

✔ Detects hate comments from text
✔ NLP preprocessing pipeline
✔ Machine Learning classification
✔ Clean and simple implementation

---

## 🧠 How It Works

🔹 **Step 1:** Text Preprocessing

* Remove noise, punctuation, stopwords

🔹 **Step 2:** Feature Extraction

* TF-IDF Vectorization

🔹 **Step 3:** Model Training

* Train ML model (Naive Bayes / Logistic Regression)

🔹 **Step 4:** Prediction

* Classify input as Hate / Non-Hate
